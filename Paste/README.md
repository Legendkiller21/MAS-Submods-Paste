Install:
- Open the zip file from the release
- Copy the `game/` folder inside the zip to the `DDLC/` folder (the folder with the DDLC exe in it) and **MERGE** the folders

Controls:
- Copy (CTRL + c)
- Paste (CTR: + v)
- Cut (CTRL + x)
- Select All (CTRL + a)
- Sub-Select (CTRL + Left Arrow/Right Arrow)
